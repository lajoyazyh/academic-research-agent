系统架构

本系统基于多 Agent 协作：研究者 Agent 负责自动搜索、下载和整理论文，生成研究笔记；撰写 Agent 基于笔记生成结构化学术综述。

使用方法

  • 在研究控制台创建新会话，输入主题，选择交互式/快速模式。
  • 交互模式下，系统会生成关键词方案供您确认，然后逐步搜索文献，最后生成综述。
  • 快速模式一键完成全部过程，结果自动保存并加入收藏夹。
  • 在历史记录页面查看以往综述和论文PDF，可加入收藏。
  • 使用AI 对话助手与文献知识问答。

数据来源

系统集成了 arXiv、Semantic Scholar、Crossref、OpenAlex 等多个学术数据库，根据领域自动选择最佳来源。

常见问题

Q: 为什么没有搜到论文？
A: 尝试使用英文关键词，或调整研究主题的泛化程度。
