# Portfolio Project Notes

## Short Description

Academic Research Agent is an interactive AI research assistant that turns a research topic into a traceable literature-review workspace.

## Long Description

The system combines planning, tool-based paper search, RAG note generation, deep literature analysis, and Markdown review drafting. It supports both a manual checkpoint workflow and a one-click automatic workflow. Users can inspect traces, edit intermediate artifacts, configure Skills, and ask a cross-session Copilot questions over previous research workspaces.

## Feature Bullets

- Plan + ReAct + Reflexion Agent pipeline
- Paper search over arXiv, Crossref, OpenAlex, and Semantic Scholar
- RAG-based structured notes from abstracts and PDFs
- Editable analysis cards for comparison, lineage, and research gaps
- Markdown literature review generation and revision
- Tool traces, error handling, and Skill fallback observability
- Cross-session knowledge Copilot
